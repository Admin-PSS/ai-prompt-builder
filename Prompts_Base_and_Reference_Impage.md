Base and Reference Image : Core Concept (Very Important)

You usually have:

Image A (Base Image) → main identity / person
Image B (Reference Image) → what you want to copy (style, pose, dress, etc.)

👉 Your prompt must explicitly say who provides what

🔑 Golden Rule

“Preserve identity from Image A, apply features from Image B”

🧩 Prompt Structure (Multi-Image)
Use Image A as the base image and primary subject.
Preserve the identity, face, and overall appearance of the person in Image A.

Use Image B as a reference for [pose / clothing / background / style].

Apply the [specific elements] from Image B to the person in Image A,
while maintaining natural proportions, realism, and consistency.

Ensure lighting, perspective, and shadows are matched across the final image.

Do not change the identity of the person from Image A.
🎯 Use Cases + Examples
👤 1. Face from A + Pose from B
Use Image A as the base image and keep the same person and facial identity.

Use Image B as a reference for body pose.

Apply the pose from Image B to the person in Image A, including arm and leg positioning,
while maintaining natural anatomy and balance.

Ensure the face, expression, and identity remain unchanged.
👗 2. Face from A + Dress from B
Use Image A as the base image and preserve the same person.

Use Image B as a reference for clothing.

Replace the outfit in Image A with the clothing from Image B,
ensuring realistic fabric texture, natural folds, and proper fit.

Maintain the original face, pose, and proportions.
🧍 3. Face from A + Full Style from B
Use Image A as the base image and keep the same person and identity.

Use Image B as a reference for overall style, including pose, outfit, and mood.

Apply the visual style from Image B while preserving the facial identity from Image A.

Ensure realistic blending, consistent lighting, and natural proportions.
🖼️ 4. Background from B
Use Image A as the base image and preserve the same person.

Use Image B as a reference for the background.

Replace the background in Image A with the environment from Image B,
ensuring correct perspective, lighting direction, and realistic shadows.

Keep the subject unchanged.
💄 5. Makeup from B
Use Image A as the base image and keep the same person.

Use Image B as a reference for makeup style.

Apply similar makeup (lip color, eye makeup, skin finish) from Image B
while maintaining natural skin texture and identity.
🔥 6. Full Combination (Most Powerful)
Use Image A as the base image and preserve the same person and facial identity.

Use Image B as a reference for pose, outfit, and background.

Apply the pose, clothing, and environment from Image B to the person in Image A.

Ensure:
- natural anatomy and proportions
- realistic lighting and shadows
- consistent perspective and depth

Do not modify the face or identity from Image A.
Avoid distortion or artificial appearance.
⚠️ Common Mistakes
❌ Bad Prompt

“Combine these two images”

👉 Too vague → unpredictable results

❌ Missing Identity Lock

👉 Face will change

❌ Mixing Instructions

👉 Model doesn’t know which image controls what

💡 Pro Tips
1. Be Explicit About Roles
“Image A = identity”
“Image B = pose/clothing/background”
2. Apply One Thing at a Time (best quality)

Instead of everything at once:

Change pose
Then clothing
Then background

👉 Better results than doing all in one

3. Add This Line (Very Powerful)
Prioritize preserving identity from Image A over all other changes.
4. Fix Lighting Issues
Match lighting direction and color temperature between subject and background.
🧠 Advanced Template (Reusable)

Preserve the same person, facial identity, and expression.

Use Image B as a reference for [INSERT: pose / clothing / background / makeup].

Apply the selected elements from Image B to Image A,
while maintaining natural anatomy, correct proportions, and realistic alignment.

Ensure consistent lighting, shadows, perspective, and color tone.

Prioritize identity preservation from Image A above all.
Avoid distortion, unrealistic blending, or artificial appearance.
